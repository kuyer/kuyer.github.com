MongoDB导入、导出、备份和恢复操作

MongoDB导出

导出命令：mongoexport

参数：
--help 说明
-d 数据库名
-c collection名称
-o 导出的文件名称
--type 导出的文件格式，默认是：json
-f 导出的字段名，如果--type为csv，则需要加上：-f '字段名'

mongoexport -u mguser -p mgpassword --authenticationDatabase admin -d db_test -c clt_test -o /tmp/clt_test.json

MongoDB导入

导入命令：mongoimport

参数：
--help 说明
-d 数据库名
-c collection名称
--file 导入的文件
--type 输出的文件格式，默认是：json
-f 导入的字段名
--headerline 如果导入的格式是csv，则可以试用第一行的标题作为导入的字段

mongoimport -u mguser -p mgpassword --authenticationDatabase admin -d db_test -c clt_test --file /tmp/clt_test.json

MongoDB数据备份

备份命令：mongodump

参数：
--help 说明
-d 数据库名
-c collection名称
-o 备份的文件名

mongodump -d db_test -c clt_test -o /tmp

MongoDB数据恢复

恢复命令：mongorestore

参数：
--help 说明
-d 数据库名
-c collection名称

mongorestore -d db_test -c clt_test /tmp/db_test/clt_test.bson
