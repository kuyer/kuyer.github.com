# 图形化

## View

View，基础视图元素，其它图形元素基于它

| 属性 | 描述 |
| id | 唯一编号 |
| name | 名称 |
| margin | 外距边缘 |
| padding | 內距边缘 |
| border | 边框 |
| position | 所在位置 |
| background | 背景 |

## Text

Text，文字

| 属性 | 描述 |
| text | 文字 |

## Edit

Edit，编辑框

| 属性 | 描述 |
| text | 文字 |
| lines | 行数 |

## Color

Color，颜色

| 属性 | 描述 |
| r | red 红色 |
| g | green 绿色 |

| a | alpha 透明度 |
