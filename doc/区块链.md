# 区块链（BlockChain）

区块链（Blockchain）是一种去中心化（Decentralization）技术的概念，其本质是一串使用密码学方法相关联产生的数据块，每一个数据块中包含过去十分钟内所有比特币网络交易信息，用于验证其信息的有效性（防伪）和生成下一个区块。

## 相同热度概念

区块链、深度学习（人工智能），量子计算
