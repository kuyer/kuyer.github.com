## 权限

* private私有类型，只能在本文件内使用，使用“-”表示

```
name string
age int
hello {
	return {
		"name": "rory",
		"age": 28
	}
}
```

* public公有方法，在本文件或引用文件中使用，使用“+”表示

```
Name string
Age int
Hello {
	return {
		"name": "rory",
		"age": 28
	}
}
```
