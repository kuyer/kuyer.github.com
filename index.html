Linux环境下搭建Git服务项目
2016-05-23 12:30

1、假如您已经安装了Git

2、假如你的Git服务主机IP为：192.168.56.110

1、搭建Git项目服务

切换到项目下：cd /home/git/projects
创建Git项目：git init --bare myproject.git

2、克隆myproject.git项目到本地

git clone app@192.168.56.110:/home/git/projects/myproject.git
