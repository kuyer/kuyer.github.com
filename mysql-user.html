MySQL用户及权限

一、创建用户

命令：create user 'username'@'host' idendified by 'password';

如：create user 'test1'@'%' idendified by '111111';

二、设置用户密码

命令：set password for 'test1'@'%'=password('222222');

三、授权

命令：grant privileges on databasename.tablename to 'username'@'host';

如：grant insert,update,select on dbtest1.tuser to 'test1'@'%';#给test1用户赋dbtest1数据库中tuser表的新增、修改及查询权限

如：grant all on dbtest1.* to 'test1'@'%';#给test1用户赋dbtest1数据库的所有权限

grant privileges on databasename.tablename to 'username'@'host' with grant option;#设置该用户可以给其它用户授权

四、查看用户权限

命令：show grants for 'test1'@'%';

五、撤销用户权限

命令：revoke privilege on databasename.tablename from 'username'@'host';

如：revoke insert,update on dbtest1.* from 'test1'@'%';

六、删除用户

命令：drop user 'username'@'host';
