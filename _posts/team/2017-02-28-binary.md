## 十进制

> 487 = 4*100 + 8*10 + 7*1 = 4*10^2 + 8*10^1 + 7*10^0

## 二进制

### 二进制转换成十进制

> 10101 = 1*2^4 + 0*2^3 + 1*2^2 + 0*2^1 + 1*2^0 = 1*16 + 0*8 + 1*4 + 0*2 + 1*1 = 21

### 十进制转换称二进制

> 21 = 21/2 => 10_1 -> 10/2 => 5_0 -> 5/2 => 2_1 -> 2/2 => 1_0 -> 1 = 10101
