# MySQL函数

MySQL提供了字符、数字、日期、条件、加密及系统本身等函数

## 数学函数

* abs: 绝对值

```sql
-- 结果都为1
SELECT ABS(-1);
SELECT ABS(1);
```

* pi: 圆周率，默认显示6位小数

```sql
-- 结果为：3.141593
SELECT PI();
```
