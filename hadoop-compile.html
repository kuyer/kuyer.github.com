CentOS系统下编译Hadoop2.7.x版本

1、安装基本类库

yum -y install ncurses-devel lzo-devel zlib-devel autoconf automake libtool cmake openssl–devel gcc*

2、安装open-jdk-8

设置环境变量： vim /etc/profile
export JAVA_HOME=/usr/java/open-jdk-1.8
export CLASSPATH=.:$CLASSPATH:$JAVA_HOME/lib:$JAVA_HOME/jre/lib
export PATH=$PATH:$JAVA_HOME/bin:$JAVA_HOME/jre/bin

使环境生效： source /etc/profile

检查Java环境： java -version

3、安装protobuf

tar -zxvf protobuf-2.6.x.tar.gz
cd protobuf-2.6.x
./configure
make
make install
测试是否安装成功：protoc --version

4、安装maven

5、安装ant

6、安装hadoop

编译hadoop mvn clean package -Pdist,native -DskipTests -Dtar
编译完成后，生产的hadoop文件在./hadoop-dist/target/hadoop-2.7.x.tar.gz路径下
